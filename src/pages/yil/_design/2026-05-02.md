## Overview

This is a warm, editorial light page — the visual equivalent of a well-curated morning newsletter. White background with colored section markers that feel like tabbed dividers: each learning cluster gets its own accent that appears in the category pill, section heading underline, code chips, and card borders. The value stream quote gets a full-bleed callout panel in amber-tinted cream, making it the emotional anchor before the content sections begin. Typography is confident — large hero headline, generous spacing, clean gray text hierarchy. The page should feel like opening something curated and intentional, not scrolling a feed.

## Do's and Don'ts

### Do's
- Use each accent color consistently within its section: pill label, h2 accent bar, inline code, card borders, bold highlights
- Give the quote callout section a distinct warm background (amber-50 or a cream tint) so it reads as a special panel, not just another section
- Use the large decorative section number (low opacity, font-black) as a visual anchor behind each section heading

### Don'ts
- Don't use dark backgrounds anywhere — this is a fully light page; dark mode variants are still fine but the light-mode base should be clean and bright
- Don't use the same layout type for adjacent sections — vary between cards, prose, and steps to create visual rhythm
- Don't let the references section feel like an afterthought — use the accent-colored category pills to tie each link back to its section
