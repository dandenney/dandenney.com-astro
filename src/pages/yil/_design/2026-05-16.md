## Overview

The Bugatti reference is the most restrained luxury-auto brand surface in the category: pure black canvas, white uppercase display type at weight 400 (never bold), wide letter-spacing as the sole form of emphasis, and no accent color save a single ice-blue (#c3d9f3) reserved for inline links. This visual language maps with unusual precision to a day about convergence in AI tooling — both share a "machined precision" quality: no ornamentation, no color spectacle, just the artifact held in generous dark space. The three-typeface system (display / serif body / monospace) mirrors the three-tool convergence story. Saira Condensed substitutes for Bugatti Display; Cormorant Garamond for Bugatti Text Regular; JetBrains Mono for Bugatti Monospace. The same trinity discipline applies: never mix the roles. The "photograph" that anchors a Bugatti hero band becomes the `/GOAL` command rendered at full-bleed scale — the primitive IS the visual.

## Do's and Don'ts

### Do's
- Set `/GOAL` in massive uppercase Saira Condensed at weight 400 with 4px letter-spacing — the command is the hero visual, the same way a Bugatti model name anchors a photo band
- Keep all section label tags in uppercase JetBrains Mono at ~11px with 2px letter-spacing — no fill, no pill border, just the monospace type floating above the headline
- Use the ice-blue (#c3d9f3 / text-blue-200) sparingly — on links and inline accents only; if everything glows blue, the monochrome discipline collapses

### Don'ts
- Don't bold any type — weight 400 throughout; emphasis comes from size, tracking, and family contrast, never weight
- Don't introduce any color accent beyond the single ice-blue; adding a second accent color breaks the total monochrome discipline the Bugatti reference enforces
- Don't add rounded corners to cards or containers — 0px everywhere; only pill-shaped transparent-outline buttons get the rounded treatment
